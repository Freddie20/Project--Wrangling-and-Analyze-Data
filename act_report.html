Analysing the Data¶

After wrangling, the data was stored to a CSV file named "twitter_archive_master.csv”.

For Analysis, the clean master CSV file was assigned to the variable name “df”. From the clean dataframe, there were a lot of insights and possibilities to explore. I was curious to know more about the names of the dogs rated, their dog stages and the dog’s retweets and favourites count.

To make sense of all I wanted to explore, I created a list of questions I would like to answer. See below:

Questions to answer

  1. What are the most common dog names in the rated dogs?
  2. Which of the dog stages has the highest favourites?
  3. Which of the dog stages has the highest retweets?
  4. What are the top 20 best-rated dog breeds in the #weratedogs archive?

Insights¶

Most common dog names¶

To get the most common dog names, I used the value_counts method which returns the counts of unique values of the column.

The result by default is in descending order so that the first element is the most frequently-occurring element.

Based on the findings the most common dog names in the #weratedogs archive are Charlie, Oliver, Tucker, Penny, Cooper, Lucy, Bo, Winston, Lola and Sadie.

Dog stages with the highest favourites¶

To get the dog stages with the highest favourites, I subsetted the data to show the dog stages using groupby operation and summed the data by favourite_count.

To show the descending order I sorted using the sort_values method.

Based on this, I found that the dog stage with the highest favourites is the pupper with an overall score of 1260481.0.

The top 20 best-rated dog breeds¶

To find the best-rated dog breeds, I also used the groupby to group dog breed names, using the mean of the rating(rating_numerator/rating_denominator) and sorted in descending order using the sort_values method.

From the analysis, I found that the top dog breed is clumber, followed by soft-coated_wheaten_terrier, west_highland_white_terrier, great_pyrenees, borzoi, labrador_retriever, siberian_husky, golden_retriever, pomeranian, saluki and tibetan_mastiff.
